# gamengine
A game engine for java

To install, in your build.gradle file add:
   ```gradle
   repositories { 
        jcenter()
        maven { url "https://jitpack.io" }
   }
   dependencies {
         implementation 'com.github.ONLYIWIN21:gamengine:RELEASE'
   }
   ```
